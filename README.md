## About

O projeto feito a partir da PokeAPI, conta com varias features, como:

* Find by name;
* Find by type;
* Pokemon info;
* Image change;
* User pokedex;
* Pagination;
* Animation;
* Responsivity;
* Extra info;

## Technology

* VueJS
* Bootstrap
* CSS
* JavaScript
